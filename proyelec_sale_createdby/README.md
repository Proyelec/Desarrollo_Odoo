# Proyelec - Created By on Quotations

**Version:** 17.0.1.0.0  
**Author:** AIT - Proyelec  
**License:** LGPL-3  
**Category:** Sales

## Description

Adds a **"Created By"** column with user avatar to the list view of Sales Quotations, making it easy to identify who created each quotation at a glance.

## Features

- **Created By** column with user avatar in the quotations list view
- Optional column (users can toggle it from column settings)
- Read-only field, not editable from the view

## Installation

1. Copy the module into your Odoo addons directory
2. Update the module list
3. Install **Proyelec - Created By on Quotations**

## Dependencies

- `sale`
