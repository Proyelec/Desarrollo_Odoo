# Proyelec - Personal Calendar

**Version:** 17.0.1.0.0  
**Author:** AIT - Proyelec  
**License:** LGPL-3  
**Category:** Productivity

## Description

Extends the room booking module (`proyelec_salas`) to sync room reservations with each user's personal calendar in Odoo, allowing meeting room bookings to appear directly in the standard calendar view.

## Features

- Synchronization of room bookings with `calendar.event` per user
- Personal calendar view with integrated room booking events
- Per-user access control over their own events

## Installation

1. Install `proyelec_salas` first
2. Copy the module into your Odoo addons directory
3. Update the module list
4. Install **Proyelec - Personal Calendar**

## Dependencies

- `calendar`
- `proyelec_salas`
